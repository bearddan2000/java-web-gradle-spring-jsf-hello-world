# java-web-gradle-spring-jsf-hello-world

## Description
A springboot web app with jsf support.

#W Tech stack
- java
  - springboot
  - jsf
- gradle

## Docker stack
- gradle:jdk11

## To run
`sudo ./install.sh -u`
[Available at](http://localhost)

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
